# Code of Conduct / 行为准则

Lodestar aims to be a useful, respectful open-source project.

Lodestar 希望成为一个有用、尊重人的开源项目。

## Expected Behavior / 期待行为

- Be respectful and constructive.
- Assume good intent, but accept correction.
- Focus criticism on ideas, code, docs, and behavior.
- Make the project easier for the next user.

- 保持尊重和建设性。
- 默认善意，但接受纠正。
- 批评聚焦于想法、代码、文档和行为。
- 让项目对下一个用户更容易使用。

## Unacceptable Behavior / 不接受的行为

- Harassment, threats, or personal attacks.
- Publishing private information without consent.
- Discriminatory language or behavior.
- Repeated disruption of project discussion.

- 骚扰、威胁或人身攻击。
- 未经同意公开他人隐私信息。
- 歧视性语言或行为。
- 反复扰乱项目讨论。

## Enforcement / 执行

Maintainers may remove comments, close issues, or block participants when behavior harms the
project community. If you see a problem, open a private maintainer contact path when available,
or open an issue with only the minimum public detail needed to explain the concern.

当行为伤害项目社区时，维护者可以删除评论、关闭 issue 或屏蔽参与者。如果你发现问题，可以在有私密维护者联系方式时私下联系；如果只能公开反馈，请只提供说明问题所需的最少公开信息。
